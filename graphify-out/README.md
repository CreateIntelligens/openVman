# openVman Knowledge Graph

Generated by [graphify](https://github.com/safishamsi/graphify) — 4122 nodes · 6851 edges · 288 communities

## Files

| File | Description |
|------|-------------|
| `graph.html` | Interactive graph — open in browser |
| `graph.json` | Raw graph data (nodes + edges) |
| `GRAPH_REPORT.md` | God nodes, communities, surprising connections |
| `wiki/index.md` | Agent-crawlable wiki (if generated) |

## Rebuild

```bash
# Full rebuild
python3 scripts/rebuild_graphify.py

# Incremental (only changed files)
python3 scripts/rebuild_graphify.py --update

# Generate wiki
python3 scripts/rebuild_graphify.py --wiki

# Query
python3 scripts/rebuild_graphify.py --query "TTS pipeline"
```

## Scope

Covers `brain/`, `backend/`, `frontend/` source files.
Excludes: `dist/`, `assets/`, `node_modules/`, `.superpowers/`, etc.
